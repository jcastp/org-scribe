* Welcome to Emacs Writing!

Welcome to ~emacs-writing~, a comprehensive toolkit designed to help you write novels and short stories in Emacs using Org-mode. If you're seeing this for the first time, you might feel overwhelmed by all the features available. Don't worry! This guide will walk you through everything step by step, starting from the very beginning.

** About This Guide

This tutorial is designed for writers who:
- Have just installed ~emacs-writing~
- Want to start writing right away
- Need a gentle introduction to the package's features
- Prefer learning by doing rather than reading technical documentation

You don't need to learn everything at once. This guide presents features in a logical order, starting with the essentials and gradually introducing more advanced capabilities.

** What You'll Learn

By the end of this tutorial, you'll know how to:
- Create and organize novel or short story projects
- Write in distraction-free environments
- Track your characters, locations, and plot threads
- Count words and monitor your progress
- Search and navigate your manuscript efficiently
- Use powerful linking features to connect your planning with your writing

** Prerequisites

Before starting, make sure you have:

✓ Emacs 29.1 or higher installed
✓ Org-mode 9.6 or higher (comes with Emacs)
✓ The ~emacs-writing~ package installed
✓ Basic familiarity with Emacs (opening files, basic navigation)
✓ Basic familiarity with Org-mode (headings, properties - we'll explain as we go)

** How to Use This Guide

This guide is organized into progressive lessons. Each lesson builds on the previous ones, so we recommend following them in order:

1. Start with Lesson 1 (Creating Your First Project)
2. Complete the exercises in each lesson
3. Move to the next lesson when you feel comfortable
4. Come back to review specific topics as needed

You can always refer to the [[file:00-feature-index.org][Feature Index]] for a complete list of available features.

* Quick Reference

** Essential Keybindings

| Key     | Action                  |
|---------+-------------------------|
| F8 F8   | Open Hydra menu         |
| C-c W   | Quick note capture      |
| C-c C-o | Follow link at point    |
| C-c &   | Return from link        |
| C-c C-c | Update dynamic table    |
| C-c C-x C-c | Toggle column view  |

** Common Hydra Menu Keys

From the Hydra menu (F8 F8):

| Key | Action                    | Key | Action                  |
|-----+---------------------------+-----+-------------------------|
| s   | Insert scene              | m   | Writing mode            |
| c   | Insert chapter            | f   | Focus mode              |
| n   | Capture note              | w   | Count words             |
| h   | Capture character         | 1   | Search by PoV           |
| l   | Capture location          | 2   | Search by character     |
| C   | Character linking menu    | 3   | Search by plot          |
| L   | Location linking menu     | 4   | Search by location      |
| P   | Plot thread linking menu  | q   | Quit menu               |
