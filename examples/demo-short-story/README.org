* Notes
This is a demonstration short story project for org-scribe. It showcases:
- Simple 3-section structure (Opening, Middle, Ending)
- Integrated word tracking
- Character and plot planning in notes.org
- ID-based character and location linking in scene properties
- Science fiction time paradox theme

* Available Commands

Open the main menu with =F8 F8= (=hydra-org-scribe/body=).

** Insert
| Key       | Command                       | What it does                               |
|-----------+-------------------------------+--------------------------------------------|
| =F8 F8 s= | =org-scribe-insert-scene=     | Insert a new scene heading with properties |
| =F8 F8 c= | =org-scribe-insert-chapter=   | Insert a new chapter heading               |
| =F8 F8 o= | =org-scribe-open-project-file= | Open a project file (notes, story…)       |

** Tag the current scene (everyday linking)
Point anywhere inside a scene heading and use these to assign ID links.

| Key       | Command                            | What it does                       |
|-----------+------------------------------------+------------------------------------|
| =F8 F8 p= | =org-scribe-set-pov-character=     | Set :PoV: link from character list |
| =F8 F8 h= | =org-scribe-set-scene-characters=  | Set :Characters: links             |
| =F8 F8 l= | =org-scribe-set-scene-locations=   | Set :Location: links               |
| =F8 F8 g= | =org-scribe-set-scene-plot-threads= | Set :Plot: thread links            |
| =F8 F8 j= | =org-scribe-jump-to-pov-character= | Jump to the PoV character's entry  |

** Writing modes (Workspace)
| Key       | What it does                                           |
|-----------+--------------------------------------------------------|
| =F8 F8 m= | Distraction-free writing mode                          |
| =F8 F8 f= | Focus mode (narrow to current section)                 |
| =F8 F8 e= | Editing mode (notes and manuscript side by side)       |
| =F8 F8 v= | Navigation mode (tree + outline)                       |
| =F8 F8 0= | Return to normal layout                                |

** Word counting and tracking
| Key       | Command                   | What it does                                         |
|-----------+---------------------------+------------------------------------------------------|
| =F8 F8 w= | =org-scribe-wordcount=    | Count words (=C-u= refreshes all WORDCOUNT properties) |
| =F8 F8 r= | =org-tracktable-write=    | Add a daily word count row                           |

** Search (org-ql powered)
| Key       | Command                             | What it does                      |
|-----------+-------------------------------------+-----------------------------------|
| =F8 F8 1= | =org-scribe-org-find-pov=           | Find all scenes by PoV character  |
| =F8 F8 2= | =org-scribe-org-find-character=     | Find scenes a character appears in |
| =F8 F8 3= | =org-scribe-org-find-plot=          | Find scenes by plot thread        |
| =F8 F8 4= | =org-scribe-org-find-location=      | Find scenes by location           |
| =F8 F8 5= | =org-scribe-search-todos-recursive= | Find all TODO/unwritten scenes    |
| =F8 F8 6= | =org-scribe-search-edits-recursive= | Find scenes flagged for editing   |

|------------+----------------------------------+---------------------------------|

| =F8 F8 kl= | =org-scribe-capture-location=    | Capture a new location          |
| =F8 F8 kg= | =org-scribe-capture-plot-thread= | Capture a new plot thread       |
| =F8 F8 kb= | =org-scribe-capture-object=      | Capture an important object     |
| =F8 F8 kt= | =org-scribe-capture-timeline=    | Capture a timeline event        |
| =F8 F8 kn= | =org-scribe-capture-to-file=     | Capture a free-form note        |
