** Synopsis
A simple story about Alex, who struggles with chaotic mornings and decides to
establish a better morning routine. Through various challenges and with help
from neighbor Sam, Alex learns to find balance between structure and flexibility.

This demo project showcases the features of the org-scribe package, including:
- Scene and chapter organization with metadata
- Character development tracking
- **ID-based character linking** (click character names in scene properties to jump to their profiles)
- Location and world-building
- Plot structure planning
- Word count tracking
- Timeline management
- And much more!

** Character Linking

This project demonstrates the **ID-based character linking system**:

- Click on character names in scene properties to jump to their profiles
- Navigate seamlessly between scenes and character database
- Menu-based character selection for scene properties
- Automatic ID creation for new characters

See the PoV and Characters properties in each scene heading in [[file:novel.org][novel.org]] for examples.

* Available Commands

Open the main menu with =F8 F8= (=hydra-org-scribe/body=).

** Insert
| Key     | Command                      | What it does                               |
|---------+------------------------------+--------------------------------------------|
| =F8 F8 s= | =org-scribe-insert-scene=      | Insert a new scene heading with properties |
| =F8 F8 c= | =org-scribe-insert-chapter=    | Insert a new chapter heading               |
| =F8 F8 o= | =org-scribe-open-project-file= | Open a project file (characters, plot…)    |

** Tag the current scene (everyday linking)
Point anywhere inside a scene heading and use these to set ID links in its
properties without typing them manually.

| Key     | Command                           | What it does                       |
|---------+-----------------------------------+------------------------------------|
| =F8 F8 p= | =org-scribe-set-pov-character=      | Set :PoV: link from character list |
| =F8 F8 h= | =org-scribe-set-scene-characters=   | Set :Characters: links             |
| =F8 F8 l= | =org-scribe-set-scene-locations=    | Set :Location: links               |
| =F8 F8 g= | =org-scribe-set-scene-plot-threads= | Set :Plot: thread links            |
| =F8 F8 j= | =org-scribe-jump-to-pov-character=  | Jump to the PoV character's entry  |

** Display toggles
| Key      | Command                       | What it does                                        |
|----------+-------------------------------+-----------------------------------------------------|
| =F8 F8 LC= | =org-scribe-column-view-toggle= | Toggle org-column-view (shows clean names, not IDs) |
| =F8 F8 LT= | =org-scribe-overlays-mode=      | Toggle entity tooltips in echo area (opt-in)        |

With =org-scribe-overlays-mode= active, moving point onto any =[[id:...]]= link
displays the entity's Role, Age, Goal, and Motivation in the echo area.
